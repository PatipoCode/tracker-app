Веб-додаток для відстеження особистих витрат.

## Функціонал
- Додавання витрат з описом, сумою та категорією
- Видалення витрат
- Фільтрація за категоріями
- Статистика витрат
- Збереження в localStorage
- Темна/світла тема

## Запуск
```bash
npm install
npm run dev
```

## Тести
